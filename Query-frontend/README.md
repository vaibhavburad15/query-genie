# Query Genie (AI Database Chatbot)

This is a React + Vite + TypeScript project that allows users to interact with databases using natural language.
The frontend is styled with Tailwind CSS and shadcn-ui components.

## Features
- Natural language to SQL conversion
- JWT authentication
- MySQL database integration
- Clean UI with shadcn + Tailwind

## Technologies Used
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn-ui
- Express + MySQL (backend)

## How to Run
```sh
npm install
npm run dev
